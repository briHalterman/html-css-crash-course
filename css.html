Classes & IDs

A class lets you style many elements the same way, using a . before the name (like .note). An ID is for one unique element, using # (like #header).

Box Model

Every HTML element is a box made of four parts: content, padding, border, and margin. Understanding this helps with spacing and layout.

Margin & Padding

Padding is the space inside the border, around the content. Margin is the space outside the border, between elements.
