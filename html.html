Basic terminology and syntax

HTML uses tags to mark up content. Tags go inside angle brackets, like <p> for paragraphs, and many come in pairs with a closing tag like </p>.

Strong and emphasis

To make text bold, use the <strong> tag. For italic or emphasized text, use <em>.

Anchors and attributes

Anchors are made with the <a> tag and are used to create links. The href attribute tells the browser where the link should go.
